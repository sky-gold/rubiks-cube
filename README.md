# rubiks-cube
C++ Solver for Rubik's cube

## Команды для запуска

Скомпилировать проект
```
make build
```
Файл программы появится по пути **build/src/rubiks_cube_solver**

Запустить тесты
```
make test
```

## Пример запуска

```
build/src/rubiks_cube_solver < static/5.txt 
```
